# Project: TechEngineering IoT Chapter Expansion

## Architecture
This project is an educational repository for the course "Digital Technology for Engineering" (เทคโนโลยีดิจิทัลสำหรับวิศวกรรม), targeted at 1st-year mechanical engineering students.
The content covers IoT systems end-to-end, starting from edge sensors/actuators to microcontroller interfacing, wireless networking, application protocols, cloud storage, visualization dashboard, and basic machine learning.

The repository structure is organized as follows:
- `chapters/`: Main theoretical and practical text guides in Markdown (`README.md`).
- `labs/`: Laboratory and exercise materials.
- `assets/`: SVGs, images, and other assets for the chapters.
- `weekly-lesson-plan.md`: The weekly lesson plan for the course.

## Interface Contracts / Guidelines for Expansion
To ensure high-quality chapter content, the following expansion standard must be adhered to:
1. **Theoretical Foundations**: In-depth explanations of physical and computational concepts (e.g., how analog-to-digital converters work, MQTT topics structures, PWM duty cycle math).
2. **Real-world Examples**: Relatable engineering applications, particularly mechanical and industrial applications (e.g., vibration-based predictive maintenance, boiler temperature control, factory conveyor belts).
3. **Illustrative Code Blocks**: Compilable/syntactically correct code samples (C++ for Arduino/ESP32, Python for ML/data science, JSON for payloads).
4. **Thai Language Style**: Clear, professional, and accessible Thai language tailored for university students.
5. **Structural Integrity**: Maintain standard markdown headers and tables, ensuring consistency with the main `summary.md`.
