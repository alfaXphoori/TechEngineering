### 🎯 เกณฑ์การตัดเกรด (Grading Scale)

| เกรด | ช่วงคะแนน (ร้อยละ) | ค่าระดับคะแนน | ความหมาย |
|:---:|:---:|:---:|:---|
| **A** | 80–100 | 4.0 | ดีเยี่ยม (Excellent) |
| **B+** | 75–79 | 3.5 | ดีมาก (Very Good) |
| **B** | 70–74 | 3.0 | ดี (Good) |
| **C+** | 65–69 | 2.5 | ดีพอใช้ (Fairly Good) |
| **C** | 60–64 | 2.0 | พอใช้ (Fair) |
| **D+** | 55–59 | 1.5 | อ่อน (Fairly Poor) |
| **D** | 50–54 | 1.0 | ผ่านเกณฑ์ขั้นต่ำ (Poor) |
| **F** | 0–49 | 0.0 | ตก (Fail) |
