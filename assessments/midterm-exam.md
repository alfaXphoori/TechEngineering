**ข้อ 11.** อธิบายความหมายของ "ADC Resolution" หรือ "ความละเอียด ADC" ของ ESP32 (12-bit) พร้อมอธิบายว่าค่านี้มีผลต่อการวัดเซนเซอร์อุณหภูมิแอนะล็อกอย่างไร

*(8 คะแนน: นิยาม ADC 2 คะแนน, การคำนวณจำนวนระดับ 2 คะแนน, ผลต่อการวัด 2 คะแนน, ตัวอย่างหรือการนำไปใช้ 2 คะแนน)*

---

**ข้อ 12.** "PWM Duty Cycle" คืออะไร อธิบายวิธีที่ ESP32 ใช้ PWM ควบคุมความเร็วของพัดลมระบายความร้อน พร้อมระบุความสัมพันธ์ระหว่าง duty cycle กับความเร็วรอบพัดลม

*(8 คะแนน: นิยาม PWM 2 คะแนน, นิยาม duty cycle 2 คะแนน, ความสัมพันธ์กับความเร็ว 2 คะแนน, ตัวอย่างค่า 2 คะแนน)*

---

**ข้อ 13.** อธิบายความแตกต่างระหว่าง Pull-up Resistor และ Pull-down Resistor ในวงจรปุ่มกด ระบุว่าแต่ละแบบให้สถานะใด (HIGH/LOW) เมื่อกดและเมื่อไม่กดปุ่ม

*(8 คะแนน: Pull-up อธิบาย 3 คะแนน, Pull-down อธิบาย 3 คะแนน, ตารางสรุปสถานะ 2 คะแนน)*

---

**ข้อ 14.** อธิบายหน้าที่ของ "Flyback Diode" ในวงจรขับรีเลย์ที่ควบคุมปั๊มน้ำ ระบุว่าจะเกิดอะไรขึ้นถ้าไม่ใส่ไดโอดนี้

*(8 คะแนน: หน้าที่ 4 คะแนน, ผลเมื่อไม่มี 2 คะแนน, ตำแหน่งการต่อ 2 คะแนน)*

**ข้อ 15.** ESP32 สามารถทำงานได้ 2 โหมดหลักในระบบ Wi-Fi คือ Station Mode (STA) และ Access Point Mode (AP) อธิบายความแตกต่างระหว่างสองโหมดนี้ พร้อมยกตัวอย่างว่าในระบบ IoT งานวิศวกรรม แต่ละโหมดเหมาะกับกรณีใด

**ข้อ 17.** ระบบควบคุมเซอร์โวมอเตอร์ (SG90) ใช้สัญญาณ PWM ที่ความถี่ 50 Hz ในการกำหนดตำแหน่ง โดย pulse width 1 ms = มุม 0° และ pulse width 2 ms = มุม 180°

**ข้อ 19.** อ่านโค้ด ESP32 ต่อไปนี้แล้วตอบคำถาม:

```cpp
const int SENSOR_PIN = 34;
const int LED_PIN    = 2;
const int THRESHOLD  = 2048;

void setup() {
  Serial.begin(115200);
  pinMode(LED_PIN, OUTPUT);
  // SENSOR_PIN ไม่ต้อง pinMode เพราะเป็น ADC input
}

void loop() {
  int rawValue = analogRead(SENSOR_PIN);
  float voltage = (rawValue / 4095.0) * 3.3;
  
  Serial.print("Raw: ");
  Serial.print(rawValue);
  Serial.print("  Voltage: ");
  Serial.print(voltage, 2);
  Serial.println(" V");
  
  if (rawValue > THRESHOLD) {
    digitalWrite(LED_PIN, HIGH);
  } else {
    digitalWrite(LED_PIN, LOW);
  }
  
  delay(500);
}
```

**(ก)** โค้ดนี้ทำงานอะไร อธิบายภาพรวมการทำงาน (2 คะแนน)

**(ข)** ถ้า Potentiometer ที่ต่อกับขา 34 ถูกหมุนจนอ่านค่า `rawValue = 3000` LED จะเป็นสถานะอะไร และค่าแรงดันไฟฟ้าที่แสดงใน Serial Monitor คือเท่าไร (2 คะแนน)

**(ค)** โค้ดนี้มีปัญหา "Button Bounce" หรือ "Noise" อยู่หรือไม่ ถ้ามีควรแก้ไขอย่างไรเบื้องต้น (2 คะแนน)
