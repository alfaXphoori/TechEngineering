**ข้อ 19.** อ่านโค้ด ESP32 ต่อไปนี้แล้วตอบคำถาม:

```cpp
const int SENSOR_PIN = 34;
const int LED_PIN    = 2;
const int THRESHOLD  = 2048;

void setup() {
  Serial.begin(115200);
  pinMode(LED_PIN, OUTPUT);
  // SENSOR_PIN ไม่ต้อง pinMode เพราะเป็น ADC input
}

void loop() {
  int rawValue = analogRead(SENSOR_PIN);
  float voltage = (rawValue / 4095.0) * 3.3;
  
  Serial.print("Raw: ");
  Serial.print(rawValue);
  Serial.print("  Voltage: ");
  Serial.print(voltage, 2);
  Serial.println(" V");
  
  if (rawValue > THRESHOLD) {
    digitalWrite(LED_PIN, HIGH);
  } else {
    digitalWrite(LED_PIN, LOW);
  }
  
  delay(500);
}
```

**(ก)** โค้ดนี้ทำงานอะไร อธิบายภาพรวมการทำงาน (2 คะแนน)

**(ข)** ถ้า Potentiometer ที่ต่อกับขา 34 ถูกหมุนจนอ่านค่า `rawValue = 3000` LED จะเป็นสถานะอะไร และค่าแรงดันไฟฟ้าที่แสดงใน Serial Monitor คือเท่าไร (2 คะแนน)

**(ค)** โค้ดนี้มีปัญหา "Button Bounce" หรือ "Noise" อยู่หรือไม่ ถ้ามีควรแก้ไขอย่างไรเบื้องต้น (2 คะแนน)

### ส่วน ข — อัตนัยสั้น (แนวเฉลย)

**ข้อ 11 — ADC Resolution (8 คะแนน)**
- ADC (Analog-to-Digital Converter) แปลงสัญญาณแอนะล็อก (แรงดัน) เป็นตัวเลขดิจิทัล
- ESP32 มี ADC 12-bit → 2¹² = 4,096 ระดับ (0–4095)
- ความละเอียด = ช่วงแรงดัน / จำนวนระดับ = 3.3 V / 4095 ≈ 0.806 mV/ระดับ
- ยิ่ง bit สูง ยิ่งแยกแยะค่าแอนะล็อกได้ละเอียดกว่า (Arduino UNO มี 10-bit = 1024 ระดับ)

**ข้อ 12 — PWM Duty Cycle (8 คะแนน)**
- PWM (Pulse Width Modulation) = สัญญาณดิจิทัลที่สลับ HIGH/LOW เร็วมาก
- Duty Cycle = สัดส่วนเวลา HIGH ต่อหนึ่งคาบ (%) เช่น 50% = HIGH ครึ่งคาบ
- พัดลม: Duty Cycle 0% = หยุด, 50% = ความเร็วครึ่งสูงสุด, 100% = ความเร็วสูงสุด
- ตัวอย่าง: `ledcWrite(channel, 128)` ที่ 8-bit resolution = 50% duty cycle

**ข้อ 13 — Pull-up / Pull-down (8 คะแนน)**

| สถานะ | Pull-up | Pull-down |
|---|---|---|
| ไม่กดปุ่ม | HIGH (3.3 V ผ่าน R ขึ้น VCC) | LOW (0 V ผ่าน R ลง GND) |
| กดปุ่ม | LOW (ต่อลง GND โดยตรง) | HIGH (ต่อขึ้น VCC โดยตรง) |

- Pull-up: อ่านค่า LOW = กดปุ่ม (logic กลับ) — ใช้กับ `INPUT_PULLUP` ใน Arduino
- Pull-down: อ่านค่า HIGH = กดปุ่ม (logic ตรง) — ต้องต่อตัวต้านทานภายนอก

**ข้อ 14 — Flyback Diode (8 คะแนน)**
- รีเลย์มีขดลวด (coil) = ตัวเหนี่ยวนำ เมื่อตัด current กะทันหัน จะเกิดแรงดัน back-EMF สูง
- Flyback diode ต่อขนานกับ coil (ขาบวกต่อ VCC, ขาลบต่อ GND ฝั่ง coil)
- ทำหน้าที่ให้กระแส back-EMF วนวงจรภายใน coil แทนที่จะไหลทะลุทรานซิสเตอร์/MCU
- ถ้าไม่ใส่: แรงดันพุ่งสูงอาจทำลาย GPIO ของ ESP32 หรือทรานซิสเตอร์ขับได้

**ข้อ 15 — Station Mode vs AP Mode (8 คะแนน)**
- **Station Mode (STA):** ESP32 เชื่อมต่อเป็น client เข้ากับ Router/Access Point ที่มีอยู่แล้ว → ได้รับ IP จาก Router → เหมาะสำหรับส่งข้อมูลขึ้นอินเทอร์เน็ต/คลาวด์ (เช่น ส่งข้อมูลอุณหภูมิขึ้น ThingsBoard)
- **Access Point Mode (AP):** ESP32 สร้างเครือข่าย Wi-Fi ของตัวเอง อุปกรณ์อื่นเชื่อมต่อมาหา ESP32 → เหมาะสำหรับพื้นที่ไม่มี Router หรือต้องการ config ESP32 เบื้องต้น

### ส่วน ค — คำนวณและวิเคราะห์ (แนวเฉลย)

**ข้อ 16 — ADC Calculation (8 คะแนน)**

**(ก)** แรงดันไฟฟ้า:
```
V = (rawValue / 4095) × 3.3
V = (2850 / 4095) × 3.3
V = 0.6959 × 3.3 ≈ 2.30 V
```

**(ข)** ค่าแรงดันน้ำมัน:
```
PSI = (V / 3.3) × 500
PSI = (2.30 / 3.3) × 500
PSI = 0.6969 × 500 ≈ 348.5 PSI
```
(หรือคำนวณตรง: PSI = (2850 / 4095) × 500 ≈ 347.6 PSI)

**(ค)** ADC threshold สำหรับ 400 PSI:
```
ADC = (400 / 500) × 4095
ADC = 0.8 × 4095 = 3276
```
ดังนั้น threshold = 3276 (แจ้งเตือนเมื่อ rawValue > 3276)

**ข้อ 17 — PWM / Servo Calculation (8 คะแนน)**

**(ก)** Duty Cycle:
- Period = 1/50 Hz = 20 ms
- 0°: pulse = 1 ms → DC = (1/20) × 100% = **5%**
- 180°: pulse = 2 ms → DC = (2/20) × 100% = **10%**

**(ข)** มุม 90°:
- pulse width = 1 ms + (90/180) × 1 ms = 1.5 ms
- Duty Cycle = (1.5/20) × 100% = **7.5%**

**(ค)** ค่า 8-bit (0–255) สำหรับ 7.5%:
```
value = (7.5 / 100) × 255 ≈ 19.125 ≈ 19
```
(รับคำตอบ 19 หรือ 20 ขึ้นอยู่กับการปัดเศษ)

### ส่วน ง — อ่านโค้ด (แนวเฉลย)

**ข้อ 19 — Code Reading (6 คะแนน)**

**(ก)** โค้ดอ่านค่าจาก Potentiometer/เซนเซอร์แอนะล็อกที่ขา 34 ด้วย ADC แปลงเป็นแรงดัน แสดงใน Serial Monitor ทุก 500ms และถ้าค่า > 2048 (ครึ่งหนึ่งของ range) ให้ติด LED ที่ขา 2

**(ข)** rawValue = 3000 > THRESHOLD(2048) → **LED = HIGH (ติด)**
แรงดัน = (3000 / 4095) × 3.3 = 0.7326 × 3.3 ≈ **2.42 V**

**(ค)** โค้ดนี้ไม่มี Debounce (ปัญหา Bounce ของปุ่มกด) แต่ใช้กับ Potentiometer ซึ่งไม่มี bounce อย่างไรก็ตาม ADC อาจมี noise — แก้ได้โดย: (1) อ่านค่าหลายครั้งแล้วหาค่าเฉลี่ย (moving average) หรือ (2) ใช้ hysteresis (ตั้ง threshold ต่างกันสำหรับการเปิดและปิด)
