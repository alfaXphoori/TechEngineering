# การเตรียมความพร้อมเครื่องมือและโปรแกรมจำลอง (Simulator)

ก่อนที่เราจะเริ่มต่อวงจรจริงและเขียนโค้ดเพื่อควบคุมไมโครคอนโทรลเลอร์ สิ่งสำคัญที่สุดคือการเตรียม "เครื่องมือ" ให้พร้อม ในเนื้อหาส่วนนี้เราจะมาทำความรู้จักกับโปรแกรมจำลองการทำงาน (Simulator) ซึ่งเปรียบเสมือนห้องทดลองเสมือนจริงในคอมพิวเตอร์ของเรา

## 1. โปรแกรมจำลอง (Simulator) คืออะไร และทำไมต้องใช้?
**Simulator** คือโปรแกรมที่จำลองการทำงานของแผงวงจร ไมโครคอนโทรลเลอร์ (เช่น Arduino, ESP32) และอุปกรณ์อิเล็กทรอนิกส์ต่างๆ โดยมีข้อดีที่สำคัญคือ:
1. **ปลอดภัย ไม่ต้องกลัวพัง:** นักเรียนสามารถทดลองต่อสายไฟผิด หรือเขียนโค้ดผิดพลาดได้เต็มที่โดยไม่ทำให้อุปกรณ์ของจริงไหม้หรือเสียหาย
2. **ทดลองได้ทุกที่:** ไม่จำเป็นต้องพกพาอุปกรณ์จริงตลอดเวลา สามารถทำงานต่อที่บ้านได้
3. **ตรวจสอบข้อผิดพลาด (Debug) ได้ง่าย:** สามารถดูการไหลของกระแสไฟฟ้า หรือสถานะของตัวแปรต่างๆ ได้ชัดเจนกว่าฮาร์ดแวร์จริง

---

## 2. การใช้งาน Wokwi บนเว็บเบราว์เซอร์ (วิธีที่ 1 - สำหรับผู้เริ่มต้น)
วิธีนี้เป็นวิธีที่ง่ายและรวดเร็วที่สุด เพราะไม่ต้องติดตั้งโปรแกรมใดๆ ลงในเครื่อง

**ขั้นตอนการใช้งาน:**
1. เปิดเว็บเบราว์เซอร์และเข้าไปที่เว็บไซต์ **[wokwi.com](https://wokwi.com)**
2. แนะนำให้คลิก **Sign in** เพื่อล็อกอินด้วยบัญชี Google (โค้ดของเราจะได้ถูกบันทึกไว้ไม่สูญหาย)
3. เลื่อนลงมาที่หัวข้อ "Start a new Project" แล้วเลือกบอร์ดที่ต้องการ เช่น Arduino Uno หรือ ESP32
4. หน้าจอจะแบ่งเป็น 2 ฝั่ง:
   * **ฝั่งซ้าย:** สำหรับเขียนโค้ด (C++)
   * **ฝั่งขวา:** พื้นที่จำลองวงจร สามารถคลิกปุ่ม `+` เพื่อเพิ่มอุปกรณ์ (เช่น LED, ตัวต้านทาน) และคลิกลากสายไฟเพื่อเชื่อมต่อวงจรได้เลย
5. เมื่อเขียนโค้ดเสร็จ ให้กดปุ่ม **▶ (Play)** เพื่อรันโปรแกรม

> ⚠️ **ข้อควรระวัง:** วิธีนี้จำเป็นต้องใช้อินเทอร์เน็ตที่เสถียร หากมีผู้ใช้งานพร้อมกันจำนวนมาก เซิร์ฟเวอร์อาจทำงานหนัก ทำให้การประมวลผลโค้ด (Compile) ล่าช้าหรือค้างได้

## 3. การใช้งาน Wokwi ใน VS Code (วิธีที่ 2 - แผนสำรองและระดับมืออาชีพ)
*เนื้อหาส่วนนี้เหมาะสำหรับการทำโปรเจกต์ขนาดใหญ่ หรือใช้แก้ปัญหาในกรณีที่เว็บไซต์ Wokwi ทำงานช้า*

การใช้งานผ่าน **Visual Studio Code (VS Code)** จะช่วยลดภาระเซิร์ฟเวอร์เว็บ เพราะคอมพิวเตอร์ของเราจะเป็นผู้คอมไพล์โค้ดเอง ทำให้โปรแกรมรันได้รวดเร็วและลื่นไหลกว่ามาก

### 3.1 ขั้นตอนที่ 1: ติดตั้งส่วนขยาย (Extension)
1. เปิดโปรแกรม **VS Code** (ต้องติดตั้ง VS Code และ PlatformIO หรือ Arduino extension ไว้ล่วงหน้า)
2. ไปที่เมนู **Extensions** (หรือกด `Ctrl+Shift+X`)
3. พิมพ์ค้นหา **"Wokwi Simulator"**
4. กดปุ่ม **Install**

### 3.2 ขั้นตอนที่ 2: การขอ License (เปิดใช้งานฟรี)
1. ในหน้าต่าง VS Code กดปุ่ม `F1` (หรือ `Ctrl+Shift+P`) เพื่อเปิดแถบคำสั่ง
2. พิมพ์คำว่า `Wokwi: Request a New License` แล้วกด Enter
3. ระบบจะเปิดหน้าเว็บเบราว์เซอร์เพื่อให้เราล็อกอิน (ล็อกอินด้วยบัญชี Google ได้)
4. เมื่อล็อกอินเสร็จ License จะถูกเชื่อมต่อกับ VS Code โดยอัตโนมัติ พร้อมใช้งานทันที

### 3.3 ขั้นตอนที่ 3: การตั้งค่าโปรเจกต์และการรัน
1. เปิดโฟลเดอร์โปรเจกต์โค้ดของเราใน VS Code
2. กด `F1` พิมพ์คำว่า `Wokwi: Init` แล้วกด Enter
3. เลือกรุ่นบอร์ดที่เราต้องการใช้งาน (เช่น Arduino Uno, ESP32)
4. โปรแกรมจะสร้างไฟล์ที่ชื่อว่า `diagram.json` ขึ้นมา (ไฟล์นี้ใช้สำหรับกำหนดอุปกรณ์และสายไฟในวงจร)
5. เมื่อเขียนโค้ดและตั้งค่าวางอุปกรณ์เสร็จแล้ว ให้กด `F1` พิมพ์คำว่า `Wokwi: Start Simulator`
6. หน้าต่างจำลองวงจรจะปรากฏขึ้นมาด้านข้าง และรันโค้ดของเราด้วยความเร็วสูงสุด!

## 4. โปรแกรมทางเลือกออฟไลน์ (Offline Alternatives)
ในกรณีฉุกเฉินที่ไม่สามารถเชื่อมต่ออินเทอร์เน็ตได้เลย เรายังมีโปรแกรมทางเลือกที่สามารถติดตั้งลงเครื่องได้:
*   **SimulIDE:** โปรแกรมแจกฟรี (Open Source) ที่ไม่ต้องใช้อินเทอร์เน็ต รองรับบอร์ดตระกูล AVR (Arduino) ได้เป็นอย่างดีเยี่ยม สามารถโหลดไฟล์ `.hex` ที่คอมไพล์แล้วมารันเพื่อจำลองวงจรได้ทันที
*   **Tinkercad Circuits:** (ออนไลน์) เสถียรมากสำหรับการสอนเบื้องต้น แต่รองรับเฉพาะบอร์ด Arduino Uno และ Micro:bit ไม่รองรับ ESP32
