## วัตถุประสงค์

- อธิบายหลักการทำงานของสัญญาณดิจิทัล (Digital Signal) แบบ HIGH/LOW ได้
- ใช้คำสั่ง `digitalRead()` อ่านค่าจากปุ่มกด (Push Button) โดยใช้ตัวต้านทานภายใน (INPUT_PULLUP)
- อ่านค่าจากเซนเซอร์ตรวจจับการเคลื่อนไหว (PIR Motion Sensor) และแสดงผลผ่าน LED
- บันทึกและวิเคราะห์ข้อมูลสถานะเซนเซอร์ผ่าน Serial Monitor

## ส่วนที่ 1: อ่านค่าปุ่มกดด้วย INPUT_PULLUP (30 นาที)

### ความรู้เบื้องต้น

เมื่อตั้งค่าขา GPIO เป็น `INPUT_PULLUP` ไมโครคอนโทรลเลอร์จะต่อตัวต้านทานภายในดึงสัญญาณขึ้นไปที่ HIGH โดยอัตโนมัติ ดังนั้น:

- **ไม่กดปุ่ม** → อ่านค่าได้ `HIGH` (1)
- **กดปุ่ม** → อ่านค่าได้ `LOW` (0)

### ขั้นตอนปฏิบัติ

1. เปิด [Wokwi Simulator](https://wokwi.com/) แล้วสร้างโปรเจกต์ใหม่ เลือกบอร์ด **ESP32**
2. ต่อวงจรตามนี้:
   - ปุ่มกดขาหนึ่ง → **GPIO 15**
   - ปุ่มกดอีกขา → **GND**
   - LED ขายาว (Anode) → ตัวต้านทาน 220Ω → **GPIO 2**
   - LED ขาสั้น (Cathode) → **GND**
3. เขียนโค้ดต่อไปนี้:

```cpp
#define BUTTON_PIN 15
#define LED_PIN 2

void setup() {
  Serial.begin(115200);
  pinMode(BUTTON_PIN, INPUT_PULLUP);
  pinMode(LED_PIN, OUTPUT);
  Serial.println("=== Lab 2: Digital Sensor ===");
}

void loop() {
  int buttonState = digitalRead(BUTTON_PIN);

  if (buttonState == LOW) {
    digitalWrite(LED_PIN, HIGH);
    Serial.println("Button: PRESSED (LOW) -> LED ON");
  } else {
    digitalWrite(LED_PIN, LOW);
    Serial.println("Button: RELEASED (HIGH) -> LED OFF");
  }

  delay(200);
}
```

4. กด **Start Simulation** แล้วเปิด Serial Monitor
5. ทดลองกดปุ่มและปล่อยปุ่ม สังเกตค่าที่แสดงบน Serial Monitor และสถานะ LED

## ส่วนที่ 2: อ่านค่าเซนเซอร์ PIR (30 นาที)

### ความรู้เบื้องต้น

เซนเซอร์ PIR (Passive Infrared) ตรวจจับรังสีอินฟราเรดจากความร้อนของร่างกาย เมื่อมีคนเคลื่อนไหวผ่านจะส่งสัญญาณ HIGH ออกมาที่ขา OUT

### ขั้นตอนปฏิบัติ

1. เพิ่มเซนเซอร์ PIR ในวงจร Wokwi (ค้นหา "PIR" ในรายการอุปกรณ์)
2. ต่อวงจรเพิ่มเติม:
   - PIR ขา VCC → **3.3V**
   - PIR ขา GND → **GND**
   - PIR ขา OUT → **GPIO 14**
3. แก้ไขโค้ดให้อ่านค่าทั้งปุ่มกดและ PIR พร้อมกัน:

```cpp
#define BUTTON_PIN 15
#define LED_PIN 2
#define PIR_PIN 14
#define LED_PIR_PIN 4

void setup() {
  Serial.begin(115200);
  pinMode(BUTTON_PIN, INPUT_PULLUP);
  pinMode(PIR_PIN, INPUT);
  pinMode(LED_PIN, OUTPUT);
  pinMode(LED_PIR_PIN, OUTPUT);
  Serial.println("=== Lab 2: Button + PIR ===");
  Serial.println("Waiting for PIR to stabilize...");
  delay(2000);
  Serial.println("Ready!");
}

void loop() {
  int buttonState = digitalRead(BUTTON_PIN);
  int pirState = digitalRead(PIR_PIN);

  // ควบคุม LED ด้วยปุ่ม
  digitalWrite(LED_PIN, (buttonState == LOW) ? HIGH : LOW);

  // ควบคุม LED ด้วย PIR
  digitalWrite(LED_PIR_PIN, pirState);

  Serial.print("Button: ");
  Serial.print(buttonState == LOW ? "PRESSED" : "RELEASED");
  Serial.print(" | PIR: ");
  Serial.println(pirState == HIGH ? "MOTION" : "NO MOTION");

  delay(300);
}
```

4. รันจำลองและทดสอบโดยคลิกที่เซนเซอร์ PIR ใน Wokwi เพื่อจำลองการเคลื่อนไหว

## ส่วนที่ 3: ระบบแจ้งเตือนอัจฉริยะ (40 นาที)

### โจทย์

ออกแบบระบบที่รวมปุ่มกดและ PIR ให้ทำงานร่วมกันตามเงื่อนไข:

| เงื่อนไข | การทำงาน |
|:---|:---|
| PIR ตรวจจับการเคลื่อนไหว | LED กะพริบเตือน (blink) |
| กดปุ่ม | หยุดการเตือน (acknowledge) จนกว่าจะตรวจจับครั้งใหม่ |

### ขั้นตอนปฏิบัติ

1. พัฒนาโค้ดจากส่วนที่ 2 ให้ทำงานตามเงื่อนไขข้างต้น
2. ใช้ตัวแปร `bool alarmActive` ควบคุมสถานะการเตือน
3. เมื่อ PIR ตรวจจับได้ → ตั้ง `alarmActive = true`
4. เมื่อกดปุ่ม → ตั้ง `alarmActive = false`
5. แสดงสถานะทั้งหมดผ่าน Serial Monitor

> 💡 **คำแนะนำ:** ใช้ `millis()` แทน `delay()` สำหรับการกะพริบ LED เพื่อไม่ให้โปรแกรมหยุดรอขณะตรวจสอบปุ่ม
