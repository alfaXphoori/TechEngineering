ใบงานปฏิบัติการนี้เป็นส่วนหนึ่งของรายวิชา **เทคโนโลยีดิจิทัลสำหรับวิศวกรรม (Digital Technology for Engineering)** สำหรับนักศึกษาหลักสูตรวิศวกรรมเครื่องกล ชั้นปีที่ 1
ในใบงานนี้ เราจะมาเรียนรู้วิธีการสร้างแอปพลิเคชันบนสมาร์ทโฟนและเว็บเพจ เพื่อควบคุมอุปกรณ์และติดตามสถานะเซนเซอร์ผ่านอินเทอร์เน็ตของสรรพสิ่ง (IoT) โดยใช้แพลตฟอร์ม **Blynk** ร่วมกับบอร์ด ESP32 บนโปรแกรมจำลอง **Wokwi**

## วัตถุประสงค์

- สามารถประยุกต์ใช้ Virtual Pin ในการส่งข้อมูลเซนเซอร์และรับคำสั่งควบคุม LED หรือรีเลย์ได้

## ส่วนที่ 1: การตั้งค่าแพลตฟอร์ม Blynk (Blynk Platform Setup) (20 นาที)

Blynk เป็นแพลตฟอร์ม IoT ที่ช่วยให้เราสร้างแอปพลิเคชันได้ง่าย ๆ โดยไม่ต้องเขียนโค้ดฝั่งแอปพลิเคชันเองทั้งหมด

1. ไปที่เว็บไซต์ [Blynk.cloud](https://blynk.cloud/) แล้วทำการสมัครสมาชิก (Sign Up) หรือเข้าสู่ระบบ
2. ไปที่เมนู **Templates** แล้วคลิก **+ New Template**
   - **Name:** `Lab14_ME_IoT`
   - **Hardware:** `ESP32`
   - **Connection Type:** `WiFi`
   - คลิก **Done**
3. ไปที่แท็บ **Datastreams** เพื่อสร้างช่องทางการรับส่งข้อมูล (Virtual Pins) โดยคลิก **+ New Datastream** -> **Virtual Pin** แล้วตั้งค่าตามตารางด้านล่างนี้:

| ชื่อข้อมูล (Name) | ขาเสมือน (Virtual Pin) | ประเภทข้อมูล (Data Type) | ช่วงค่า (Min-Max) | การใช้งาน |
| :--- | :--- | :--- | :--- | :--- |
| LED Control | V1 | Integer | 0 - 1 | ควบคุมหลอดไฟ (ปุ่มกด/สวิตช์) |
| Temp Sensor | V2 | Double | 0 - 100 | แสดงค่าอุณหภูมิ (เกจ/กราฟ) |

4. ไปที่เมนู **Search (แว่นขยาย)** -> **Devices** -> **+ New Device** -> **From template** 
5. เลือก Template ที่เพิ่งสร้าง (`Lab14_ME_IoT`) แล้วคลิก **Create**
6. ระบบจะแสดงหน้าต่างที่มี **Auth Token** (คัดลอกเก็บไว้ เพราะต้องนำไปใส่ในโค้ด)

## ส่วนที่ 2: การออกแบบหน้าจอควบคุม (Web Dashboard & Mobile App) (15 นาที)

เราจะสร้างหน้าจอสำหรับสั่งงานและดูค่าเซนเซอร์แบบเรียลไทม์

1. ในหน้ารายละเอียดของ Device ให้ไปที่แท็บ **Web Dashboard**
2. ลากวิดเจ็ต (Widget) จากแถบด้านซ้ายมาวางบนหน้าจอ:
   - **Switch:** คลิกที่เฟือง (Settings) ของ Switch เลือก Datastream เป็น `LED Control (V1)`
   - **Gauge:** คลิกที่เฟือง ตั้งค่า Datastream เป็น `Temp Sensor (V2)`
   - **Chart:** คลิกที่เฟือง ตั้งค่า Datastream เป็น `Temp Sensor (V2)`
3. คลิกปุ่ม **Save and Apply** ที่มุมขวาบน
4. (ตัวเลือกเสริม) โหลดแอปพลิเคชัน **Blynk IoT** บนสมาร์ทโฟน (iOS/Android) ล็อกอินด้วยบัญชีเดียวกัน และปรับแต่งหน้าจอแอปมือถือโดยใช้วิดเจ็ตที่อ้างอิงกับ Datastream V1 และ V2 เช่นเดียวกัน

## ส่วนที่ 3: การเขียนโปรแกรม ESP32 บน Wokwi (ESP32 Programming on Wokwi) (30 นาที)

1. เปิดเว็บไซต์ [Wokwi.com](https://wokwi.com) และสร้างโปรเจกต์ ESP32 ใหม่
2. เพิ่มอุปกรณ์: หลอด LED 1 ดวง (ต่อขา 2) และ Potentiometer (ต่อขา 34) เพื่อใช้จำลองเป็นเซนเซอร์วัดอุณหภูมิ
3. ใน Wokwi ให้ไปที่แท็บ `Library Manager` (ไอคอนหนังสือทางซ้ายมือ) และกดปุ่ม `+` เพื่อเพิ่มไลบรารี **Blynk** 
4. คัดลอกและวางโค้ดด้านล่างนี้ลงใน `sketch.ino` (อย่าลืมเปลี่ยน Token ของตัวเอง)

```cpp
// ใส่ข้อมูล BLYNK_TEMPLATE_ID, BLYNK_TEMPLATE_NAME และ BLYNK_AUTH_TOKEN ของนักศึกษา
// ข้อมูลเหล่านี้ดูได้จากหน้า Device Info บนเว็บ Blynk
#define BLYNK_TEMPLATE_ID "TMPLxxxxxx"
#define BLYNK_TEMPLATE_NAME "Lab14_ME_IoT"
#define BLYNK_AUTH_TOKEN "Your_Auth_Token_Here" // เปลี่ยนเป็น Token ของคุณ

#define BLYNK_PRINT Serial
#include <WiFi.h>
#include <WiFiClient.h>
#include <BlynkSimpleEsp32.h>

// ตั้งค่า Wi-Fi สำหรับ Wokwi Simulator
char ssid[] = "Wokwi-GUEST";
char pass[] = "";

const int ledPin = 2;   // ขาของหลอด LED (สมมติเป็นรีเลย์ควบคุมปั๊มน้ำหรือพัดลม)
const int potPin = 34;  // ขา Potentiometer (จำลองการอ่านค่าเซนเซอร์)

BlynkTimer timer;

// ฟังก์ชันนี้จะถูกเรียกอัตโนมัติเมื่อมีการกดปุ่มสวิตช์ในแอป Blynk (V1)
BLYNK_WRITE(V1) {
  int pinValue = param.asInt(); // อ่านค่าจาก V1 (จะเป็น 0 หรือ 1)
  digitalWrite(ledPin, pinValue); // สั่งเปิด-ปิด LED ตามค่าที่รับมา
  
  Serial.print("LED Status: ");
  Serial.println(pinValue == 1 ? "ON" : "OFF");
}

// ฟังก์ชันสำหรับอ่านค่าเซนเซอร์แล้วส่งไปที่ Blynk (V2)
void sendSensor() {
  int rawValue = analogRead(potPin);
  
  // แปลงค่า analog (0-4095 ของ ESP32) ให้เป็นอุณหภูมิจำลอง (0-100 องศาเซลเซียส)
  float temp = (rawValue / 4095.0) * 100.0;
  
  Blynk.virtualWrite(V2, temp); // ส่งค่าไปยัง Virtual Pin V2
  
  Serial.print("Temperature: ");
  Serial.println(temp);
}

void setup() {
  Serial.begin(115200);
  pinMode(ledPin, OUTPUT);
  
  // เริ่มต้นเชื่อมต่อ Wi-Fi และ Blynk Cloud
  Blynk.begin(BLYNK_AUTH_TOKEN, ssid, pass);
  
  // ตั้งเวลาให้ฟังก์ชัน sendSensor ทำงานทุก 2000 มิลลิวินาที (2 วินาที)
  timer.setInterval(2000L, sendSensor);
}

void loop() {
  Blynk.run(); // ให้ Blynk ทำงานรับส่งข้อมูล
  timer.run(); // ให้ Timer ทำงาน
}
```
